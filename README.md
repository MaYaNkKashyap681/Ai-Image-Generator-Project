# Ai-Image-Generator-Project
An Ai Image Generator App.

<h3>Technology Used</h3>
<ul>
<li>React</li>
<li>Express</li>
<li>MongoDb</li>
<li>OpenGPT Api</li>
</ul>

<h3>Installation Guide</h3>

1). Open Client Folder and server Folder in the Terminal.<br/>
2). Type npm install in both the terminals.<br/>
3). Create .env file in your server folder and place the keys of cloudinary and opengpt dalle ai.<br/>
4). type npm run dev in client terminal and nodemon start in server terminal. <br/>
